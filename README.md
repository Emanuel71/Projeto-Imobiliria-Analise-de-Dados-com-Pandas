# Projeto Imobiliária: Análise Exploratória e Tratamento de Dados

Projeto prático focado na limpeza, transformação e análise estatística de uma base real de aluguéis no Rio de Janeiro. O objetivo é simular a atuação de um Analista de Dados, transformando dados brutos em inteligência para precificação e suporte a decisões imobiliárias.

## 🎯 Objetivo do Projeto
* **Saneamento de Dados:** Detectar e tratar anomalias, duplicatas e valores ausentes (Data Cleaning).
* **Segmentação:** Identificar padrões de preços por tipologia de imóvel e localização.
* **Inteligência de Negócios:** Criar visualizações para facilitar a interpretação de métricas pelas áreas de negócios.

## 🛠️ Stack Tecnológico
* **Python:** Linguagem principal.
* **Pandas:** Manipulação, limpeza e transformação estruturada dos dados.
* **Matplotlib:** Geração de visualizações e gráficos.
* **Trello (Kanban):** Gestão ágil das etapas do projeto.

## ⚙️ Pipeline de Desenvolvimento
1. **Diagnóstico Inicial:** Extração de arquivos `.csv` e mapeamento da estrutura e tipos de dados.
2. **Limpeza e Tratamento:** Remoção de nulos (`.dropna()`), inputação de valores (`.fillna()`) e padronização de registros inconsistentes.
3. **Engenharia de Atributos:** Criação de novas variáveis (numéricas e categóricas) para enriquecer o contexto da análise.
4. **Análise Exploratória (EDA):** Agrupamento, filtragem e cálculo de estatísticas descritivas (médias, frequências e distribuições).
5. **Visualização e Entrega:** Geração de gráficos comparativos e exportação da base tratada para consumo gerencial.
